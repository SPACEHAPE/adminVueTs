
## 技术栈
vue3 + typeScript + ehcarts + element-plus + vite


## 安装步骤

```
git clone https://github.com/SPACEHAPE/adminVueTs   //下载代码到本地

npm install                               
// 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn

// 开启服务器，浏览器访问 http://localhost:3000
npm run dev

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build
```
